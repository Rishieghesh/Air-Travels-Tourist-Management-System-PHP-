Air-Travels_Tourist-Management-System in PHP-----Installation Steps(Configuration)

1.Download and Unzip the file on your local system.
2.Copy Air-Travels_Tourist-Management-System folder inside root directory (for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/HTML)

Database Configuration

Open PHPMyAdmin
Create Database tms
Import database tms.sql (available inside the zip package)
Open Your browser put inside browser “http://localhost/Air-Travels_Tourist-Management-System”
